# primeiro-pr

Projeto de teste para aprender o fluxo de Pull Request com o Claude Code.

## Instalação

Clone o repositório:

```bash
git clone https://github.com/rianchamon/primeiro-pr.git
```

## Uso

Entre na pasta do projeto após cloná-lo:

```bash
cd primeiro-pr
```

A partir daqui você já pode editar os arquivos e abrir seus próprios Pull Requests.
